# MindEase

A mental health platform developed as part of an international Erasmus+ project in collaboration with Technikum Wien University, for the 3rd semester at Fontys University of Applied Sciences.

## Features

- **Role-Based Access:** Four distinct user roles — patient, therapist, admin and student intern
- **Therapist Matching:** Patients are matched with therapists based on a mental health assessment taken during registration
- **Document Verification:** Therapists must submit credentials verified by admins before becoming active
- **Self-Help Exercises:** Categorized exercises tailored to specific mental health conditions
- **Mental Health Information:** Detailed explanations of various mental health issues
- **Profile Management:** User profile pages for all roles

## Tech Stack

- **Language:** Java, JavaScript
- **Backend:** Spring Boot, Maven
- **Frontend:** React
- **Database:** PostgreSQL
- **Containerization:** Docker
- **Version Control:** Git
